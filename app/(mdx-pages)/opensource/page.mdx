These are some of the open source projects I've worked on that I regularly use in my own projects.

| Project                                                                                  | Description                                                                                                                                                       | Blog Post                                                                                                              |

| [DataQueue.dev](https://dataqueue.dev)                                                   | a lightweight job queue for Node.js/TypeScript projects, backed by PostgreSQL.                                                                                    | [Introducing DataQueue](blog/dataqueue-dev-background-job-queue-postgresql-typescript)                                 |

| [env-to-t3](https://github.com/nicnocquee/env-to-t3)                                     | CLI to generate T3 Env code from env file                                                                                                                         | [Stop using environment variable directly in your TypeScript code](/blog/stop-using-environment-variable-directly)     |
